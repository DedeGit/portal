# News Portal Website

Website portal berita berbasis web.

## Features
- Menampilkan daftar berita
- Kategori berita
- Admin panel CRUD
- Responsive design

## Tech Stack
- PHP Native
- MySQL
- Bootstrap

## Installation
1. Clone repository
2. Import database
3. Jalankan di localhost

## Database Setup
Import file `database/newsportal.sql` ke phpMyAdmin.
